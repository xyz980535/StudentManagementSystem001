# 学生信息管理系统（Student Management System）

这是一个使用 **C 语言** 编写的简易学生信息管理系统，通过 **TXT 文件** 实现数据的存储与读取。  
系统可完成学生信息的添加、查询、修改、删除等基本功能，适合初学者学习文件读写与结构体应用。

---

## 🧩 功能简介

- 📥 **添加学生信息**（姓名、学号、成绩等）  
- 🔍 **查询学生信息**（根据学号或姓名）  
- ✏️ **修改学生信息**  
- 🗑️ **删除学生信息**  
- 💾 **将数据保存至文本文件（StuAllInfo.txt、StuPWInfo.txt、TeaPWInfo.txt）**  
- 📂 **从文本文件读取数据到内存**

---

## 🛠️ 技术实现

- **编程语言**：C  
- **数据存储**：文本文件（`students.txt`）  
- **开发环境**：Dev-C++、Code::Blocks、VSCode 或其他支持 C 的编译器  
- **主要知识点**：
  - 文件读写（`fopen`, `fprintf`, `fscanf`, `fclose`）  
  - 结构体（`struct`）  
  - 函数模块化设计  
  - 循环与条件控制语句
